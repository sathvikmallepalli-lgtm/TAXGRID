## Common GSTIN Validation Errors

### Error 1: Invalid Checksum
- **Cause**: Last digit doesn't match calculated checksum
- **Solution**: Verify GSTIN with supplier

### Error 2: Invalid State Code
- **Cause**: First 2 digits don't match valid state code
- **Solution**: Check state code (01-38)

### Error 3: Invalid PAN Format
- **Cause**: Characters 3-12 don't follow PAN pattern
- **Solution**: Verify PAN structure (5 letters, 4 digits, 1 letter)

### Error 4: Invalid Entity Code
- **Cause**: 13th character not valid entity type
- **Solution**: Should be 1-9, A-Z

### Error 5: Invalid Registration Number
- **Cause**: 14th character not A-Z
- **Solution**: Must be alphabet for registration number
